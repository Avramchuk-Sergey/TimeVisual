# TimeVisual

TimeVisual — визуализатор времени.

Идея приложения:

> Время уходит. Ты сам решаешь, чем его наполнить.

## Текущий статус

Работает:

- текущее время
- песочные часы
- меню
- система экранов
- создание нового периода
- расчет секунд до выбранной даты и времени
- экран активного периода
- обратный отсчет

## Технологии

- HTML
- CSS
- JavaScript
- Local project without build tools
